Este repositório apresenta uma análise estrutural do **Google Web Graph** utilizando o dataset `web-Google` do **Stanford Network Analysis Project (SNAP)**. O objetivo é investigar se essa rede apresenta propriedades de **rede de escala livre (scale-free network)** através da análise da distribuição de graus.

## 🧠 Sobre o Dataset

O dataset **web-Google** representa a estrutura de **hiperlinks da web** coletada pelo Google em 2002.

Na modelagem em grafo:

- **Nó (vertex):** representa uma página web  
- **Aresta dirigida (edge):** representa um hiperlink de uma página para outra  

Esse tipo de rede é fundamental para estudar **estrutura da web, centralidade e padrões de conectividade**.

---

## 🔎 Objetivo da Análise

O objetivo deste projeto é investigar se a rede apresenta comportamento de **rede de escala livre (Scale-Free Network)**.

Redes de escala livre possuem uma característica importante:

> A distribuição de graus segue uma **lei de potência**.
